# 🎬 محمّل سينمانا

**برنامج يحمّل المسلسلات والأفلام بضغطة زر واحدة من موقع [سينمانا شبكتي](https://cinemana.shabakaty.com).**

تلصق رابط أي حلقة، فيجلب البرنامج **كل المواسم وكل الحلقات** تلقائياً ويحمّلها **بالتسلسل**
(حلقة تكتمل ثم تبدأ التالية) مع ملفات الترجمة، بجودة **1080p افتراضياً** — كل ذلك بضغطة زر واحدة.

> 💡 للاستخدام الشخصي لمشترك مدفوع، لمشاهدة المحتوى دون اتصال بالإنترنت.

---

## ✨ المزايا

- 🎬 **جلب تلقائي** لكل المواسم والحلقات من رابط واحد.
- ⏬ **تحميل تسلسلي** حلقة تلو الأخرى، أو **تحميل متوازٍ** بعدّة اتصالات مع **تقسيم كل ملف لأجزاء** (حتى 32 جزءاً مثل برامج التحميل الاحترافية).
- 💬 تحميل **الترجمة** (عربية / إنجليزية / كلتاهما) بصيغتي `srt` و `vtt` بجانب كل حلقة.
- 🔁 **استكمال تلقائي** بعد انقطاع الإنترنت (عبر طلبات Range).
- 🔌 **النجاة من إطفاء الجهاز**: الحالة تُحفظ على القرص؛ أعِد التشغيل ليُكمل من حيث توقّف.
- 💾 **حفظ الجلسة واستعادتها:** عند إعادة الفتح يُستعاد الرابط والمجلد والجودة والاختيار وقائمة المسلسل، مع عرض **نسبة ما تم تحميله لكل حلقة** (بدون إنترنت).
- ⏯️ **أزرار إيقاف مؤقت/استئناف عامة** لكل التحميلات دفعةً واحدة، بالإضافة إلى التحكّم بكل حلقة عبر الزر الأيمن (إيقاف/استئناف/إلغاء).
- 🔎 **فحص مجلد الحفظ** ليخبرك بالحلقات التي حمّلتها وأيّها لم يُحمَّل بعد، مع إلغاء تحديد المكتمل تلقائياً.
- 🎚️ **اختيار الجودة** (افتراضي 1080p، مع تراجع تلقائي إن لم تتوفّر).
- 🗂️ **تبويب المكتبة:** يعرض المحتوى المُنزَّل مجمّعاً (مسلسل ← مواسم ← حلقات) مع الحالة والحجم، وزرّي **فتح** الملف و**المجلد**.
- ⚙️ **تبويب الإعدادات:** اللغة، ومجلد الحفظ والجودة والتزامن الافتراضية — تُحفَظ وتُطبَّق تلقائياً على أي مسلسل لاحق.
- 🌐 **واجهة عربية/إنجليزية** قابلة للتبديل لحظياً مع دعم الاتجاه من اليمين لليسار.

## 📦 المتطلبات

- **Python 3.9+** (مُختبَر على 3.13) — أو استخدم النسخة المستقلّة (.exe) دون أي تثبيت.
- المكتبات: `requests` و `pywebview`. للتثبيت:
  ```
  pip install -r requirements.txt
  ```
  على ويندوز تعمل الواجهة عبر متصفّح **Edge WebView2** المدمج مع النظام — لا حاجة لأي تثبيت إضافي.

---

## ▶️ التشغيل

**الواجهة الرسومية (تصميم Aurora):** انقر نقرتين على `تشغيل.bat`، أو:

```
python app.py
```

**سطر الأوامر (اختياري):**

```
python -m cinemana "https://cinemana.shabakaty.com/video/ar/1200938" --quality 1080 --dest "D:\Series"
python -m cinemana 1200938 --list          # عرض الحلقات فقط
```

---

## 🖱️ طريقة الاستخدام

1. الصق **رابط المسلسل** في الحقل العلوي واضغط **جلب الحلقات**.
2. اختر **مجلد الحفظ** و**الجودة** (الافتراضي 1080p).
3. اضغط **ابدأ التحميل** — وتابع التقدّم في الجدول والسجل.
4. **للإيقاف المؤقت** اضغط **إيقاف مؤقت ⏸**، و**للاستئناف** اضغط **استئناف ▶**.
5. **عند إعادة الفتح لاحقاً** (حتى بعد إطفاء الكمبيوتر): يُستعاد كل شيء تلقائياً مع نسبة كل حلقة — اضغط **استئناف ▶** ليُكمل من حيث توقّف.

---

## 🛠️ بناء نسخة مستقلّة (.exe)

لإنتاج نسخة تعمل **بدون تثبيت Python** — انقر نقرتين على `build_exe.bat`، أو:

```
python -m PyInstaller cinemana.spec --clean --noconfirm
```

يُنتَج مجلّد `dist\Cinemana Downloader\` فيه `Cinemana Downloader.exe` مع ملحقاته؛ وزّع المجلّد كاملاً (ملف الـ exe وحده لا يكفي).

> 📥 يمكنك تنزيل النسخة الجاهزة من صفحة [الإصدارات (Releases)](../../releases).

## ⚙️ كيف يعمل الاستكمال؟

روابط سينمانا موقّعة وتنتهي صلاحيتها بعد ساعات، لذا يجلب البرنامج رابطاً جديداً لكل حلقة لحظة تحميلها،
ويتابع التنزيل من حجم ملف `.part` الحالي عبر ترويسة `Range`. ولا يُحوَّل الملف إلى `.mp4` نهائي إلا بعد اكتماله بالكامل.
